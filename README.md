# FK Sloboda Tuzla

## O projektu
Ovaj projekat je posvećenu fudbalskom klubu Sloboda Tuzla, jednom od najstarijih i najznačajnijih fudbalskih klubova u Bosni i Hercegovini. Cilj je bio stvoriti modernu digitalnu platformu koja dostojno reprezentuje klupsku tradiciju dugu više od vijeka.

Stranica služi kao centralno mjesto za navijače, fokusirajući se na najvažnije aspekte kluba: najavu predstojećih utakmica na stadionu Tušanj, predstavljanje aktuelnog igračkog kadra, te čuvanje uspomene na legende koje su gradile istoriju Slobode.
